# Danfoss Compressor Comparison Tool

A modern, interactive comparison tool for Danfoss scroll compressors against Copeland equivalents. Built with React, Vite, and Tailwind CSS.

## Features
- **Dynamic Comparison Engine**: Real-time side-by-side technical comparison.
- **Global Search**: Find models across all series (H, S, MLZ, DSH, MT/MTZ) instantly.
- **Series Insights**: Context-aware technical summaries and advantage highlights.
- **Vite + Tailwind v4**: Blazing fast development and modern styling.

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
